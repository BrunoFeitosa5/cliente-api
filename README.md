# 🚀 Cliente API - Flask CRUD com JWT

Projeto backend desenvolvido em Flask com foco em API REST, autenticação JWT e CRUD completo de clientes.

---

## 📌 Funcionalidades

- Cadastro de clientes
- Listagem de clientes
- Atualização de dados (PUT)
- Remoção de clientes (DELETE)
- Login de usuário
- Autenticação com JWT (em evolução)
- Hash de senha com bcrypt
- Banco de dados SQLite

---

## 🧠 Tecnologias utilizadas

- Python
- Flask
- Flask-JWT-Extended
- SQLite
- bcrypt
- HTML/CSS (frontend simples)
- JavaScript (consumo da API)
